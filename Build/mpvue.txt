快速创建
1、检查nodejs和npm版本
	node -v
	npm -v
2、修改源
	npm set registry https://registry.npm.taobao.org/
3、全局安装vue-cli
	npm install --global vue-cli
4、创建基于mpvue-quickstart的项目
	vue init mpvue/mpvue-quickstart qs-mpvue
5、安装依赖
	cd qs-mpvue
	npm install
	npm run dev
6、导入项目
	D:\qs-mpvue\dist\wx
7、编译运行
	npm start
