一、Jenkins安装
	1、使用docker.io/jenkins
		docker pull jenkins
		docker run --name Jenkins -d -p 8083:8080 -p 50000:50000 jenkins
	2、使用tomcat jenkins.war
		docker pull openjdk:8-jdk-alpine
		docker build --build-arg JAR_FILE=qs.jar -t qs:1.0 .
		docker create --name qs -p 8383:8383 qs:1.0
		docker start qs
