常用命令
	1、管理员权限
		su命令
			重装系统后需要设置root密码
				sudo passwd root
	2、配置环境变量
		编辑环境变量	vim /etc/profile
		当前终端中生效	source /etc/profile
	3、更新缓存
		apt-get update
	4、更新软件
		apt-get upgrade
	5、自动解决依赖
		apt-get install -f
	6、删除没有使用的依赖
		apt-get install autoremove
	7、安装.deb文件
		dpkg -i sougoupinyin.deb
	8、运行.sh文件
		./jmeter.sh
	9、运行.jar文件
		java -jar restclient.jar


命令行工具

	2、autojump
		apt-get install autojump
